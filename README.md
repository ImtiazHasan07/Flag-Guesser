# Flag Guesser Game

Welcome to the Flag Guesser Game project, a Node.js and Express web application where users try to guess countries based on their flags. The game provides an interactive way to test and improve your geography knowledge.

## Features

* Random country flag generation
* Multiple-choice guessing system
* Score tracking across rounds
* Simple and responsive web interface
* Server-side logic using Express

## Prerequisites

Before running the Flag Guesser Game, ensure you have the following:

* Node.js (v14.0 or higher recommended)
* npm (Node Package Manager)

## Installation

Follow these steps to get your Flag Guesser Game up and running:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/imtiazhasan07/flag-guesser.git
   cd flag-guesser
   ```

2. **Install dependencies:**
   Use npm to install required packages:

   ```bash
   npm install
   ```

## Usage

To launch the Flag Guesser Game and start the Express server, run the following command:

```bash
node .
```

Then open your browser and visit:

```
http://localhost:80
```

Start guessing flags and try to achieve the highest score possible.

## Project Structure

* `server.js` / `app.js` – Main Express server file
* `public/` – Frontend assets (HTML, CSS, JS)
* `views/` – Server-rendered pages (if applicable)
* `data/` – Country/flag dataset
